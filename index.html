Ask AI Daily Automation — Poster Archive

Daily Slack poster images for build-with-dhiraj/ask-ai-daily-automation.

This branch hosts PNGs rendered by the daily eval and digest pipelines. Filenames are date-stamped (YYYY-MM-DD.png); files older than 90 days are pruned automatically.

Surfaces

  • posters/scoreboard/ — Rubric Scoreboard daily posters
  • posters/digest/ — Daily Digest posters

URL pattern

https://build-with-dhiraj.github.io/ask-ai-daily-automation/posters/{surface}/{YYYY-MM-DD}.png

Operational notes

  • GitHub Pages serves this branch from the repo root.
  • New uploads take ~30–60s to be reachable after push.
  • Retention: 90 days, enforced by .github/workflows/prune-posters.yml on main.
  • Do not commit anything else to this branch. It is rendered output only.
